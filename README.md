# ![alt text](https://cfconventions.org/Data/media/images/logo.png) A-Pro-CFiler

Objective: make A-Profiles files CF compliant


## installation
  - with *poetry*

    `poetry install .`
  - with *pip*

    `pip install .`

## run conversion script
  - with *poetry*
    - `poetry shell`
    - `python -m main {data_path}/**/AP*.nc`
